# Snack API

Snack API is a simple snack api with static values.

## Installation

```bash
git clone https://github.com/caioitiroko/snack-api
```

Run at the root of Snack API

```bash
npm install
```

## Documentation
https://simple-snack-api.herokuapp.com/docs/

## Usage

```bash
npm start
```
